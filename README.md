# NASA News
* It is an IOS app developed using Swift that allows users to explore the latest stories and developments from NASA. You can view a comprehensive breakdown of each news article.
 

## Technologies
* MVC 
* Unit Testing
* URLSession
* CoreData
* KingFisher
